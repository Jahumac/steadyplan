Why it exists

Built as a personal project after serious money stress and debt/IVA. The turning point was visibility: seeing the full picture clearly, then staying engaged with it.

Shared for self-hosters who want local control, repairability and backups.

Ways to use it

Run

Server/NAS/desktop. Docker/Unraid friendly.

Use

Desktop/tablet/phone browser.

Install

PWA where supported.

Keep GitHub for releases/issues; keep the docs human-readable on the site.
