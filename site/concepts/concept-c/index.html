Long-term finance planning, self-hosted.

SteadyPlan

Face the numbers. Keep control. Plan steadily.

SteadyPlan is a self-hosted UK finance planning app. It helps you model accounts, debts, contributions, goals, investments and retirement scenarios in one private place — without handing your financial life to another platform.

Install quickstart Read docs

Why it exists

SteadyPlan started as a personal project after serious money stress and debt/IVA. Visibility — facing the numbers clearly — was the turning point. The project is shared for people who value self-hosting, local control, and systems you can back up and repair.

Quickstart (Docker)

Persist /app/data. That’s where your SQLite database and keys live. 

docker pull ghcr.io/jahumac/steadyplan:latest
docker run -d \
  --name steadyplan \
  --restart unless-stopped \
  -p 8000:8000 \
  -v /path/to/steadyplan-data:/app/data \
  ghcr.io/jahumac/steadyplan:latest

Before risky updates: take a backup. See Backups & restore.

Data & backups (trust boundaries)

Database

Local SQLite under your mounted data directory.

Exports

Per-user JSON exports you can store privately.

Instance backups

Whole-instance SQLite backups and server-level recovery.

Ways to use it

Run on your own hardware

Server/NAS/desktop. Docker/Unraid friendly.

Use from any device

Desktop/tablet/phone browser. Your phone does not host the app.

Install as a web app

PWA where supported by your browser.

LAN/VPN is the safest default. If you expose SteadyPlan publicly, use HTTPS on a trusted reverse proxy.
