Why SteadyPlan exists

SteadyPlan started as a personal project. After serious money stress and debt, the turning point wasn’t a trick — it was visibility. I needed one calm place to see the full picture — accounts, debts, savings, investments, goals and retirement plans — without sending all of that data to another finance platform. SteadyPlan is the tool I built to keep doing that.

Read the story Start with the docs

What it is (and isn’t)

  • A planning and visibility tool for long-term decisions.
  • Self-hosted, with a local SQLite database and your own backup routine.
  • Not a bank, broker, or financial adviser. No promises — projections are estimates based on your inputs.

Safest default is LAN/VPN-only. If you expose SteadyPlan publicly, use HTTPS on a trusted reverse proxy and configure it intentionally.

Install

Docker/Unraid-first. Persist /app/data, then you’re running.

Backups & restore

JSON exports and whole-instance SQLite backups — plus what to back up for real recovery.

Privacy & data ownership

Local database, no cloud sync, and clear notes on optional external lookups.
